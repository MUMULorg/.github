## 📌 프로젝트 개요
**MUMUL**은 익명 또는 닉네임으로 자유롭게 질문하고  
**공개/비공개 답변**을 선택할 수 있는 **AMA(Ask Me Anything) 기반 커뮤니티 플랫폼**입니다.

> 사람과 사람을 잇는 커뮤니티 — 개인적인 고민, 지식 공유, 취미, 관계, 익명 의견 등  
> 누구나 **부담 없이 질문하고 답할 수 있는 온라인 공간**을 목표로 합니다.

## 🔧 기술 스택

### Backend
- Java 17 · Spring Boot 3.0.5 · Spring Security · JWT · Gradle
- Gmail API (알림 이메일) · JPA ORM

### Frontend
- React 16.8 · Axios · React Router · Swiper · moment.js

### Infra
- AWS Elastic Beanstalk · Nginx · HTTPS
- GitHub Actions (CI/CD)

---

## 📦 핵심 기능

| 기능 | 설명 |
|------|------|
| 익명/닉네임 질문 | 익명 토끼🐰 또는 본인 닉네임으로 질문 등록 |
| 공개/비공개 답변 | 공개 또는 질문자에게만 보이는 비공개 답변 |
| 팔로우 | 관심 스페이스 팔로우 및 소식 받기 |
| 이메일 알림 | 내 스페이스에 질문/답변 등록 시 Gmail 자동 알림 |
| 스페이스 중지 | 질문 수신/팔로우 기능 OFF하여 쉬는 모드 |
| 반응형 UI | PC/모바일 전부 최적화 |

## 🏁 프로젝트 의의
MUMUL은 단순한 익명 커뮤니티가 아니라  
**서로의 질문에 응답하고 공감하며 성장을 도울 수 있는 디지털 AMA 플랫폼**을 목표로 합니다.

“무엇이든 물어보세요(무물)”이라는 표현처럼  
누구나 가볍게 질문하고, 진심을 담아 답할 수 있는 공간을 추구합니다.

✨ 스페이스, 팔로우, 이메일 알림, 익명 UX까지 —  
**지속 가능한 커뮤니티 경험을 고민한 프로젝트입니다.**
